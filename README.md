## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Jevmis/qa-engineering-portfolio.git
cd qa-engineering-portfolio
```

Install dependencies:

```bash
pip install mkdocs-material
```

Serve locally:

```bash
mkdocs serve
```

Build the static site:

```bash
mkdocs build
```
