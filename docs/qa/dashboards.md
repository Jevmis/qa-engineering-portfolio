# 📊 QA Dashboards

## Automated QA Reporting

One of my biggest contributions as a QA Lead has been building automated reporting dashboards using Google Sheets.

These dashboards provide real-time visibility into testing progress, defect trends, sprint health, and release readiness.

---

## Dashboard Features

<div class="grid cards" markdown>

- :material-chart-line:

    ### Test Execution Dashboard

    Track Passed, Failed, Blocked and Not Executed tests live.

- :material-bug:

    ### Bug Dashboard

    Defect trends, severity distribution and aging.

- :material-percent:

    ### Pass Rate Metrics

    Automatic percentage calculations.

- :material-view-dashboard:

    ### Executive Dashboard

    High-level KPIs for stakeholders.

</div>
